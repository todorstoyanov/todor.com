The Jackpot Classic Edition

The goal of this project was to develop software which visualizes the current jackpot in gambling clubs equipped with the Amatic Industries .Multi-Game Classic Edition. machines. All the machines are connected with one controller, which calculates the jackpot on the basis of the data sent by the machines. The jackpot value is accessible from the controller through serial interface. The software runs on a Linux PC connected with the controller with serial cable, it read the data from the serial port and visualizes it on big plasma screen. The visualization was done using Macromedia Flash while the backend was written in PHP with the DIO extension enabled. It is not so common to use PHP as standalone application, but it doses the job very well without any problems.
